# Utility Box

A small, fun Python CLI toolkit made for learning. This project is just a little playground to get comfortable with Python, CLI apps, and Git.

## What’s inside

- `main.py` — entry point with a clean interactive menu
- `calculator.py` — basic arithmetic operations
- `diceroller.py` — roll a regular or custom die
- `pw_gen.py` — password generator placeholder for future additions

## Why this exists

This is a beginner-friendly project, built to practice:

- Python scripting
- CLI interaction using `questionary`
- basic project structure
- version control with Git

## How to use it

1. Install dependencies:

```bash
pip install -r requirements/requirements.txt
```

2. Run the app:

```bash
python main.py
```

3. Pick a tool from the menu and follow the prompts.

## Notes
